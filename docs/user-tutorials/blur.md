# Blur

*Soften an object for depth, glow, or focus.*

## Use case
Blur softens an object's edges. Use it to push a background out of focus, add a
soft glow behind a shape, create a frosted look, or draw the eye to the sharp parts
of your design. A simple slider for instant depth.

## Relevant for
- Any selected object.
- Soft backgrounds, glows, shadows, and depth-of-field looks.

## How to test it
1. Select an object.
2. Open the **Effects** tab in the right panel.
3. Raise the **Blur** slider — the object turns soft and hazy.
4. Lower it back to 0 to make it crisp again.

## Related properties
- **Blur** — amount of softening, from 0 (sharp) upward.
- Pairs well with **Opacity** and **Layering** for depth effects.
