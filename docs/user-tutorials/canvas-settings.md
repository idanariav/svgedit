# Canvas Settings

*Set the size and shape of your drawing surface.*

## Use case
The canvas is your page. Set its width and height to match where the artwork will
go — a square for social posts, 16:9 for slides, or a custom size for print. Change
the background color to preview your design on light or dark. The quick aspect
presets resize the page in one click.

## Relevant for
- Starting a new project at the right dimensions.
- Resizing the page to fit a target format.

## How to test it
1. Open the **Canvas** settings popover in the top toolbar.
2. Type a new **Width** and **Height** — the page resizes.
3. Click an aspect preset like **16:9** or **1:1** for an instant shape.
4. Change the **Background** color to preview your design.
5. Click **Apply** to keep it, or **Reset** to revert.

## Related properties
- **Width / Height** — exact page size.
- **Aspect presets** — 4:5, 5:4, 16:9, 1:1, 4:3.
- **Background color** — the canvas surface color.
