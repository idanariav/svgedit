# Color Shift

*Recolor objects fast without picking new colors.*

## Use case
Color Shift nudges an object's existing colors all at once — change the hue, make
it richer or duller, lighter or darker, or more transparent. Perfect for trying
color variations, creating matching sets in different tints, or toning artwork to
fit a palette. Reset any time to return to the original.

## Relevant for
- Any selected object (works on fill, stroke, or both).
- Recoloring whole shapes or testing color variations.

## How to test it
1. Select a colored object.
2. Open the **Effects** tab in the right panel.
3. Drag the **Hue** slider — the colors rotate to new ones.
4. Adjust **Saturation** and **Lightness** for richer or softer tones.
5. Click **Reset** to return to the original colors.

## Related properties
- **Hue** — rotate the colors.
- **Saturation** — more vivid or more gray.
- **Lightness** — lighter or darker.
- **Transparency** — fade the colors.
- **Fill / Stroke toggles** — choose what the shift affects.
- **Reset** — back to the original colors.
