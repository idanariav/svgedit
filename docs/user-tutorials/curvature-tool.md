# Curvature Tool

*Drop points and get a flowing, smooth curve automatically.*

## Use case
The Curvature tool draws smooth, flowing curves through the points you click — no
fiddling with handles. Use it for graceful arcs, rolling hills, ribbons, and clean
organic lines. Mix in sharp corners wherever you want a crisp change of direction.

## Relevant for
- Smooth curves that pass cleanly through chosen points.
- Illustrations, flow shapes, and decorative lines.

## How to test it
1. Click the Curvature tool in the left toolbar.
2. Click several points across the canvas — a smooth curve threads through them.
3. Hold **Shift** while clicking to make a sharp corner instead of a smooth bend.
4. Finish the curve and switch to Select to style it.

## Related properties
- **Curve style** — Catmull-Rom, B-spline, or Spiro (each curves a little differently).
- The result is a path, so **path editing** applies for later tweaks.
