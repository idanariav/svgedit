# Drop Shadow

*Lift objects off the page with a realistic shadow.*

## Use case
A drop shadow makes an object look like it floats above the page. Use it for cards,
buttons, badges, and floating panels — or push the settings for a long, dramatic
shadow. Control the direction, distance, softness, color, and strength.

## Relevant for
- Any selected object.
- Cards, buttons, and any design that wants depth.

## How to test it
1. Select an object.
2. Open the **Effects** tab in the right panel.
3. Set a **Shadow length** above 0 — a shadow appears.
4. Change the **Angle** to move the shadow around the object.
5. Raise the **Blur** for a softer shadow, and tweak **Opacity** and **Color**.

## Related properties
- **Angle** — direction of the shadow (clockwise from the top).
- **Length** — how far the shadow is offset.
- **Blur** — how soft or sharp the shadow is.
- **Opacity** — how strong the shadow looks.
- **Color** — the shadow's tint.
- **Remove** — delete the shadow.
