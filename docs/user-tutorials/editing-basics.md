# Editing Basics

*Undo mistakes, duplicate objects, and delete what you don't need.*

## Use case
These everyday actions keep you moving. Undo and redo let you experiment without
fear. Clone copies an object instantly so you can build repeating elements. Delete
removes anything you don't want. The essentials you'll reach for constantly.

## Relevant for
- **Undo / Redo** — any time after you've made a change.
- **Clone / Delete** — any selected object (or several at once).

## How to test it
1. Draw a shape, then press **Ctrl+Z** to undo it.
2. Press **Ctrl+Y** to redo it back.
3. Select the shape and click **Clone** (or press **D**) — a copy appears.
4. Select a shape and press **Delete** to remove it.

## Related properties
- **Undo** — step back through your changes.
- **Redo** — step forward again.
- **Clone / Duplicate** — make an instant copy of the selection.
- **Delete** — remove the selection.
