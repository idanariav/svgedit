# Export

*Turn your drawing into a shareable image or document.*

## Use case
When your artwork is ready, export it as an image or document to use anywhere — a
PNG for the web, a JPG for email, a WebP for small file sizes, or a PDF for print
and documents. One step takes your design out of the editor.

## Relevant for
- A finished or in-progress drawing you want to save as a picture.
- Sharing, embedding, or printing your work.

## How to test it
1. Open the **Export** option from the menu.
2. Choose a format — PNG, JPG, WebP, or PDF.
3. Confirm — the file is generated and downloaded.
4. Open the file to check it matches the canvas.

## Related properties
- **Format** — PNG, JPG, WebP, or PDF.
- To keep an editable copy instead of an image, see **Open, Save & Import**.
