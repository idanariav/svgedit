# Eyedropper

*Copy a look from one object to another in one click.*

## Use case
The Eyedropper grabs the color and style from an object on the canvas and applies
it where you want. Use it to match colors exactly, reuse a style you already
created, and keep a design consistent without remembering hex codes.

## Relevant for
- Copying fill, stroke, and style from an existing object.
- Keeping colors consistent across a drawing.

## How to test it
1. Style one shape the way you like it.
2. Pick the Eyedropper tool.
3. Click a different shape — it takes on the sampled fill, stroke, and style.

## Related properties
- Copies **fill, stroke, and style** in a single action.
- Works alongside **Fill & Stroke Color** and **Style Presets**.
