# Fill & Stroke Color

*Give every shape its color — inside and outline.*

## Use case
Color is what brings a design to life. **Fill** colors the inside of a shape;
**stroke** colors its outline. Set either to "none" for outline-only or fill-only
looks. Change the canvas background, or grab a color fast from the palette.

## Relevant for
- Any shape, text, or path.
- Setting the canvas background color.

## How to test it
1. Select a shape.
2. Click the **Fill** swatch and pick a color — the inside changes.
3. Click the **Stroke** swatch and pick a different color — the outline changes.
4. Set Fill to **none** — only the outline remains.
5. Click a swatch in the **palette** for a one-click color.

## Related properties
- **Fill** — inside color (or none).
- **Stroke** — outline color (or none).
- **Background** — canvas color behind everything.
- **Palette** — quick-pick swatches.
- For multi-color fills, see **Gradients**. To copy a color from the canvas, see
  the **Eyedropper**.
