# Flip & Rotate

*Mirror and spin objects to any angle.*

## Use case
Flip an object to mirror it left-right or top-bottom — great for symmetrical
designs, facing arrows, or reflections. Rotate to tilt text, angle an arrow, or
spin a shape to exactly the degree you want.

## Relevant for
- Any selected object.
- Mirrored layouts, reflections, and angled artwork.

## How to test it
1. Select an object.
2. Click **Flip Horizontal** — it mirrors left to right.
3. Click **Flip Vertical** — it mirrors top to bottom.
4. In the right panel, set the **Rotation** angle (for example 45°) — it tilts.
5. Or drag the round rotate handle above the selection.

## Related properties
- **Flip Horizontal** — mirror left/right.
- **Flip Vertical** — mirror top/bottom.
- **Rotation angle** — exact tilt, from −180° to 180°.
