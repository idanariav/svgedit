# Fonts

*Choose the perfect typeface, including Google Fonts.*

## Use case
The right font sets the tone — elegant, playful, technical, or bold. Pick from the
built-in typefaces, or browse the Font Library to add hundreds of Google Fonts.
Downloaded fonts are saved, so your text keeps its look everywhere.

## Relevant for
- Any text object.
- Branding, headings, and stylized labels.

## How to test it
1. Select a text object.
2. Open the **Font family** dropdown and pick a different typeface — the text updates.
3. Open the **Font Library** to browse more fonts.
4. Search or filter by category, then pick a font to add and apply it.

## Related properties
- **Font family** — the active typeface, with live previews.
- **Font Library** — browse and add Google Fonts by name or category.
- See **Text** for size, weight, and spacing options.
