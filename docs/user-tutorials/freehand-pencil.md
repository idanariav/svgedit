# Freehand Pencil

*Sketch naturally, like drawing on paper.*

## Use case
The Pencil lets you draw freehand by dragging. Use it for signatures, sketches,
handwritten notes, doodles, and loose organic lines that would be slow to build
point by point.

## Relevant for
- Quick, hand-drawn marks and sketches.
- Anyone with a mouse, trackpad, or stylus.

## How to test it
1. Click the Pencil tool, or press **Q**.
2. Press and drag across the canvas — the line follows your movement.
3. Release to finish the stroke.
4. Switch to Select and change its stroke color or width.

## Related properties
- The result is a path, so all **path editing** applies (double-click to refine).
- **Stroke width, color, dash** — see Stroke Properties and Fill & Stroke Color.
