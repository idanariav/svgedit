# Grid & Snapping

*Line everything up perfectly with a guiding grid.*

## Use case
A grid gives you guidelines to draw against, and snapping locks objects to it so
everything lines up cleanly. Switch to an isometric or perspective grid for 3D-style
drawings, pixel art, technical diagrams, and tidy layouts.

## Relevant for
- Precise layouts and aligned drawings.
- Isometric, pixel, and technical artwork.

## How to test it
1. Open the **Grid** settings popover in the top toolbar.
2. Turn on **Show grid** — guidelines appear on the canvas.
3. Turn on **Snap** and draw — your shapes lock to the grid lines.
4. Change the grid **shape** to isometric and watch the lines tilt.

## Related properties
- **Show grid** — display the guidelines.
- **Snap to grid** — lock objects to grid points.
- **Grid shape** — square, isometric, triangle, or perspective.
- **Step** — spacing between grid lines.
- **Color** — the grid line color.
