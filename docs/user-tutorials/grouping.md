# Grouping

*Bind objects together so they move and scale as one.*

## Use case
Group related objects so they behave like a single unit — move, resize, or rotate
them together without disturbing their arrangement. Perfect for a finished icon, a
labeled diagram, or any cluster you want to keep intact. Ungroup any time to edit
the pieces again.

## Relevant for
- **Grouping** needs two or more objects selected.
- **Ungrouping** is available when a group is selected.

## How to test it
1. Select two or more objects.
2. Click **Group**, or press **G** — one box now surrounds them all.
3. Drag the group — every piece moves together.
4. Click **Ungroup** — the objects become independent again.

## Related properties
- **Group** — combine the selection into one unit.
- **Ungroup** — break a group back into separate objects.
- **Group title** — give the group a name/label.
- **Unlink** — turn a reused (linked) copy into an independent, editable object.
