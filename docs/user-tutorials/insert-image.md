# Insert Image

*Bring photos and logos into your drawing.*

## Use case
Drop a photo, logo, or screenshot into your canvas and build around it. Use it for
backgrounds, product shots, reference images, and brand marks. Load from a file on
your computer or paste in a web address.

## Relevant for
- Adding existing raster images (PNG, JPG, etc.) to a design.
- Any layout that mixes drawn shapes with real pictures.

## How to test it
1. Click the Image tool in the left toolbar.
2. The **Insert image** dialog opens.
3. Either choose a file from your computer or paste an image URL.
4. Confirm — the image appears on the canvas.
5. Select it and drag a corner handle to resize.

## Related properties
- **Width / Height** — exact image size in the right panel.
- **URL** — the image source address, editable later.
- Clip, mask, blur, and shadow all work on images too.
