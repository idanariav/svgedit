# Layering (Z-Order)

*Control what sits on top and what hides behind.*

## Use case
When objects overlap, layering decides which one shows in front. Bring a label
above its background, push a shadow behind a shape, or send a backdrop all the way
back. Essential for badges, stacked cards, and any overlapping artwork.

## Relevant for
- Any object that overlaps another.
- One object at a time, or a multi-selection.

## How to test it
1. Draw two overlapping shapes with different colors.
2. Select the back one.
3. Click **Bring to Front** — it jumps above the other.
4. Click **Send Backward** — it drops one step down.

## Related properties
- **Bring to Front** — all the way to the top.
- **Bring Forward** — up one step.
- **Send Backward** — down one step.
- **Send to Back** — all the way to the bottom.
- **Switch layers** — swap the stacking order of exactly two selected objects.
- For organizing big drawings, see also the **Layers Panel**.
