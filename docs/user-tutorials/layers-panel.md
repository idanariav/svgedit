# Layers Panel

*Organize complex drawings into stacked, named layers.*

## Use case
Layers keep big drawings manageable. Put the background on one layer, the content
on another, and labels on top. Hide a layer while you work, rename layers so you
remember what's where, and move objects between them. Like transparent sheets
stacked on top of each other.

## Relevant for
- Drawings with many objects that benefit from grouping into sheets.
- Separating background, content, and annotations.

## How to test it
1. Open the **Layers** tab in the right panel.
2. Click **New Layer** — a fresh layer appears in the list.
3. Draw something — it lands on the active layer.
4. Toggle a layer's visibility to hide and show it.
5. Drag a layer up or down to change which one sits on top.

## Related properties
- **New / Delete / Rename layer** — manage the layer list.
- **Move layer up / down** — change stacking order.
- **Visibility toggle** — hide or show a whole layer.
- **Move to layer** — send the selected object to another layer.
