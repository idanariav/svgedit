# Line

*Straight strokes for dividers, links, and connectors.*

## Use case
Draw straight lines for dividers, underlines, axes, and simple connectors between
shapes. Add arrowheads at either end to turn a line into a pointer or a flow arrow.

## Relevant for
- Drawing a single straight segment between two points.
- Diagrams, timelines, charts, and callouts.

## How to test it
1. Click the Line tool, or press **L**.
2. Drag from a start point to an end point — a straight line appears.
3. Adjust the endpoint numbers in the right panel for an exact length.
4. Open the marker options and add an arrowhead to one end.

## Related properties
- **Start point (X1, Y1)** / **End point (X2, Y2)** — exact endpoints.
- **Stroke width, dash style, line cap** — see Stroke Properties.
- **Markers / arrowheads** — see Markers & Arrowheads.
