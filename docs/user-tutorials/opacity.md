# Opacity

*Make objects see-through for layering and subtle accents.*

## Use case
Opacity controls how see-through an object is. Lower it for soft overlays,
watermarks, faded backgrounds, or to let a layer beneath show through. A subtle way
to add depth and blend elements together.

## Relevant for
- Any selected object.
- Overlays, watermarks, and layered, blended looks.

## How to test it
1. Select an object.
2. Lower the **Opacity** slider toward 0%.
3. The object fades; whatever sits behind it shows through.
4. Raise it back to 100% to make it solid again.

## Related properties
- **Opacity** — 0% (invisible) to 100% (solid).
- Combine with **Layering** to control what shows through.
