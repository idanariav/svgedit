# Open, Save & Import

*Keep editable copies and bring in other artwork.*

## Use case
Save your work as an SVG file so you can reopen and keep editing it later — colors,
shapes, and layers all stay editable (unlike an exported image). Open an existing
SVG to pick up where you left off, or import another SVG to combine drawings.
Drag-and-drop makes it instant.

## Relevant for
- Saving an editable copy of your project.
- Reopening past work, or merging in another drawing.

## How to test it
1. Choose **Save** — an SVG file of your drawing downloads.
2. Choose **Open** and select an SVG — it loads onto the canvas.
3. Choose **Import** (append) to add another SVG on top of your current one.
4. Or drag an SVG file straight onto the canvas.

## Related properties
- **Open** — load an SVG to edit.
- **Save** — store an editable SVG copy.
- **Import / Append** — add another SVG into the current drawing.
- **Clear** — start a fresh, empty canvas.
- For images or PDFs instead of editable SVG, see **Export**.
