# Select & Move

*Point, click, and rearrange your whole drawing.*

## Use case
The Select tool is your home base. Use it to pick objects, drag them into place,
and resize them by their corner handles. It's how you build a clean layout —
nudging a logo, lining up icons, or stretching a box to fit.

## Relevant for
- Any object already on the canvas.
- One object, or many at once (multi-select).
- Switching out of any drawing tool to start editing instead of creating.

## How to test it
1. Click the arrow (Select) tool in the left toolbar, or press **S**.
2. Click an object — a box with handles appears around it.
3. Drag the object to a new spot.
4. Drag a corner handle to resize it.
5. Hold **Shift** and click a second object — both are now selected and move together.

## Related properties
- **Move** — drag the body of the selection.
- **Resize** — drag a corner or side handle.
- **Multi-select** — Shift+click, or drag a box around several objects.
- Position and size also show as editable numbers in the right-side panel.
