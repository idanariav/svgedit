# Shape Library

*A drawer of ready-made shapes — drop them in and go.*

## Use case
Skip drawing from scratch. The Shape Library offers ready-made shapes — arrows,
stars, banners, symbols, and more — organized by category. Great for flowcharts,
icons, decorations, and quick mockups.

## Relevant for
- Adding a common shape without drawing it by hand.
- Diagrams, icon sets, and fast prototyping.

## How to test it
1. Click the Shape Library tool in the left toolbar.
2. A pop-up shows shapes grouped into categories.
3. Browse a category and click a shape.
4. It drops onto the canvas, ready to move and resize.

## Related properties
- Once placed, a shape behaves like any drawn object: fill, stroke, transform, and
  effects all apply.
