# Stroke Properties

*Shape the outline — thickness, dashes, and corners.*

## Use case
The stroke is an object's outline, and its details set the mood. A thick stroke
feels bold; a dashed stroke suggests a cut line or a path; rounded joins and caps
look friendly, while sharp ones look technical. Tune them for borders, arrows,
dividers, and diagrams.

## Relevant for
- Any shape, line, or path that has an outline.
- Borders, arrows, dashed guides, and connectors.

## How to test it
1. Select an object with a visible outline.
2. Raise the **Stroke width** — the outline gets thicker.
3. Change the **Dash style** to dashed or dotted.
4. Switch the **Line cap** to round and watch the line ends soften.

## Related properties
- **Stroke width** — outline thickness.
- **Dash style** — solid, dotted, dashed, dash-dot, dash-dot-dot.
- **Line join** — how corners meet: miter, round, or bevel.
- **Line cap** — how line ends finish: butt, round, or square.
- See also **Fill & Stroke Color** for the outline color.
