# Style Presets

*Save a look once, reuse it everywhere.*

## Use case
A style preset bundles a set of looks — fill, stroke, and more — under a name you
choose. Apply it to any object in one click, and update the preset to restyle
everything that uses it at once. Perfect for keeping a consistent, on-brand design
across a big drawing.

## Relevant for
- Any object you want to style consistently.
- Designs that reuse the same look in many places.

## How to test it
1. Style an object the way you like it.
2. In the right panel, open the **Style preset** picker and save it under a name.
3. Select another object and choose that preset — it takes on the saved look.
4. Update the preset and watch every object using it change together.

## Related properties
- **Apply preset** — stamp a saved style onto the selection.
- **Save / update preset** — store the current look, or refresh an existing one.
- **Delete preset** — remove a saved style.
- Each apply is a single, undoable step.
