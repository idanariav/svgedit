# Text

*Add headings, labels, and captions in any style.*

## Use case
Add words to your design — titles, labels, captions, callouts, and watermarks.
Style them bold or italic, underline them, change the font and size, and tune the
spacing for a polished, on-brand look. A perspective tilt can give text a 3D feel.

## Relevant for
- Any place that needs words: headings, labels, annotations.
- Editing existing text (double-click it to retype).

## How to test it
1. Click the Text tool, or press **T**.
2. Click on the canvas and type.
3. Click away to finish, then re-select the text.
4. In the right panel, toggle **Bold** and **Italic**, then change the **Font size**.
5. Open the **Font family** dropdown and pick a different typeface.

## Related properties
- **Bold / Italic / Underline / Strikethrough / Overline** — text styling.
- **Font family** — the typeface (plus the Font Library for Google Fonts).
- **Font size** — text height.
- **Text anchor** — align text left (start), center (middle), or right (end).
- **Letter spacing / Word spacing** — gaps between letters and words.
- **Perspective X / Y** — tilt the text for a 3D effect.
- See also **Fonts** for adding new typefaces.
