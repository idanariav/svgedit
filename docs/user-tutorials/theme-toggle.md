# Theme Toggle

*Switch the editor between light and dark.*

## Use case
Work the way that's easiest on your eyes. Light theme suits bright rooms and print
work; dark theme is gentler at night and helps when designing for dark interfaces.
The switch changes the editor's look, not your artwork.

## Relevant for
- Anyone using the editor — a personal comfort setting.

## How to test it
1. Click the theme toggle in the top toolbar.
2. The editor switches between light and dark.
3. Click again to switch back.

## Related properties
- Affects the editor interface only; your drawing's colors stay the same.
- Your choice is remembered for next time.
