# Wireframe & SVG Source

*Peek under the hood — outlines only, or the raw code.*

## Use case
Wireframe mode strips everything to plain outlines, so you can see structure and
overlaps without colors getting in the way — handy for untangling a busy drawing.
The SVG source view shows the underlying code, where you can paste in artwork or
make precise edits by hand.

## Relevant for
- Reviewing the structure of a complex drawing (wireframe).
- Advanced users who want to read or edit the raw SVG (source).

## How to test it
1. Click the **Wireframe** toggle in the top toolbar — colors vanish, outlines remain.
2. Click it again to return to the normal view.
3. Click **Edit SVG Source** — the raw code opens.
4. Make a small change and apply it, or just close the view.

## Related properties
- **Wireframe** — outline-only display toggle.
- **Edit SVG Source** — view and edit the underlying SVG code.
